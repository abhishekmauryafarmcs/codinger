Test Cases Table Structure

Columns:

  • id - int(11) NOT NULL
  • problem_id - int(11) NOT NULL
  • input - text NOT NULL
  • expected_output - text NOT NULL
  • is_visible - tinyint(1) NULL
  • created_at - timestamp NOT NULL

Visibility Column:

Column 'is_hidden' does NOT exist

Column 'is_visible' exists
